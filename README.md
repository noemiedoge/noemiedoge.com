## Requirements
 * Docker

## Installation

```bash
$ cp docker-compose.override-example.yml docker-compose.override.yml
$ vim docker-compose.override-example.yml

$ docker-compose up -d --build
```

Then go to [localhost:8080](http://localhost:8080).
